Google reCAPTCHA for WordPress. Currently supports:

- WordPress Login/Register
- WooCommerce Login/Register
- WP Job Manager (Submission Form)
- Claim Listing for WP Job Manager
- Gravity Forms
- Ninja Forms
- Contact Form 7

= Documentation =

Usage instructions for this plugin can be found on our documentation: [http://docs.astoundify.com/](http://docs.astoundify.com/).

== Installation ==

To install this plugin, please refer to the guide here: [http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation)

Visit Settings > reCAPTCHA and complete configuration.
